# 🛡️ Digital Defense (Defesa Digital) - Educational Game

**[Play](https://viniguirodri.github.io/Security-Playground-Digital-Defense/)**

*Leia em Português abaixo / Read in Portuguese below*

**Digital Defense** is an interactive web app created to teach information security in a practical and accessible way. The main goal of the project is to help people (especially audiences less familiar with technology, such as the elderly) identify everyday digital scams before they become victims.

## 🎯 How does it work?
The game simulates real interfaces of everyday apps. The player acts as a "digital detective" and must find the **Red Flags** hidden in the scammers' messages.

Currently, the project features the following simulations:
* **✉️ Email (Phishing):** Identifying fake senders and malicious links.
* **💬 WhatsApp (Social Engineering):** The classic "new number" scam asking for money transfers.
* **📞 SMS (Delivery Scam):** Fake customs or postal fee charges.
* **📱 Social Media (Deepfakes):** Identifying AI-generated videos making fake government program promises.

## 🚀 Technologies Used
Built with a focus on being lightweight and accessible, running directly in the browser without the need for installation:
* **HTML5** (Semantic structure)
* **CSS3** (Responsive styling simulating real mobile interfaces)
* **Vanilla JavaScript** (Game logic, interactivity, and scoring system)
* **Local Storage** (Saving player progress and coins directly in the browser)

# 🛡️ Defesa Digital - Jogo Educativo

**Defesa Digital** é um web app interativo criado para ensinar segurança da informação de forma prática e acessível. O objetivo principal do projeto é ajudar pessoas (especialmente o público com menos familiaridade tecnológica, como os idosos) a identificar golpes digitais do dia a dia antes de se tornarem vítimas.

## 🎯 Como funciona?
O jogo simula interfaces reais de aplicativos que usamos todos os dias. O jogador atua como um "detetive digital" e precisa encontrar as **Red Flags** (sinais de alerta/erros) escondidas nas mensagens dos golpistas.

Atualmente, o projeto conta com as seguintes simulações:
* **✉️ E-mail (Phishing):** Identificação de remetentes falsos e links maliciosos.
* **💬 WhatsApp (Engenharia Social):** O clássico golpe do "novo número" pedindo PIX.
* **📞 SMS (Golpe da Entrega):** Falsas cobranças de taxas alfandegárias/Correios.
* **📱 Redes Sociais (Deepfake):** Identificação de vídeos gerados por Inteligência Artificial fazendo falsas promessas de programas do governo.

## 🚀 Tecnologias Utilizadas
Projeto construído com foco em leveza e acessibilidade, rodando diretamente no navegador sem necessidade de instalação:
* **HTML5** (Estrutura semântica)
* **CSS3** (Estilização responsiva simulando interfaces mobile reais)
* **JavaScript Vanilla** (Lógica do jogo, interatividade e sistema de pontuação)
* **Local Storage** (Salvamento de progresso e moedas do jogador diretamente no navegador)
